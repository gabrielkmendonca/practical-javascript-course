Calling a JavaScript File

There are 2 ways of calling a JavaScript function. It can be either internal ou external calls.

  • For internal calling, use the <script> tag. It can be used in both <head> and <body> sections.
  • For external calling, it is possible to call a JavaScript file using the <script:src> tag. It's preferable to use in the end of the <body>, .

Script calling priority depends on the order in wich scripts are called, regardless of the section in which they are inserted

External JavaScript Advantages

  • It separates the HTML and the code
  • It makes HTML and JavaScript easier to read and maintain
  • Cached JavaScript files can speed up page loads
  • The same JavaScript can be used for more than one page
