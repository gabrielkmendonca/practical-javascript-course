Ways os JavaScript Outputs

The JavaScript can display the data in differente ways:

  • Writting into an HTML element
    • using "innerHTML" or "innerText"
  • Writing into the HTML output
    • using "document.writeln()"
  • Writing into an alert box
    • using "window.alert()"
  • Writing into the browser console
    • using "console.log()"

To access the HTML element, you can use the "document.getElementByID(id)" method. Then use "innerHTML" in order to change the HTML content, or "innerText" to change the inner text.

For debugging purposes, you can call the "console.log()" method in the browser to display data.

Last but no least, use can use the "window.print()" method on click, in order to print the content of the current window.
